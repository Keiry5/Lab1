Introducción a la Programación Web con PHP

PHP es un lenguaje de programación de servidor ampliamente usado para el desarrollo web. Es ideal para principiantes porque es fácil de aprender y se integra bien con HTML. En esta página, exploraremos estructuras de control básicas que te ayudarán a controlar el flujo de tu código.

If

La estructura if se usa para ejecutar código si una condición es verdadera. Puedes agregar else para alternativas.

<?php
$edad = 18;
if ($edad >= 18) {
    echo "Eres mayor de edad.";
} else {
    echo "Eres menor de edad.";
}
?>

Ejemplo: Verifica si un usuario puede votar basado en su edad.

Switch

La estructura switch evalúa una variable contra múltiples casos, ideal para menús o selecciones.

<?php
$dia = "Lunes";
switch ($dia) {
    case "Lunes":
        echo "Inicio de semana.";
        break;
    case "Viernes":
        echo "Fin de semana cerca.";
        break;
    default:
        echo "Día normal.";
}
?>

Ejemplo: Muestra un mensaje según el día de la semana.

While

El bucle while repite código mientras una condición sea verdadera.

<?php
$contador = 1;
while ($contador <= 5) {
    echo "Número: $contador <br>";
    $contador++;
}
?>

Ejemplo: Imprime números del 1 al 5.

For

El bucle for es compacto para repeticiones con contador conocido.

<?php
for ($i = 0; $i < 10; $i++) {
    echo "Iteración: $i <br>";
}
?>

Ejemplo: Cuenta del 0 al 9.

Foreach

El bucle foreach itera sobre arrays o colecciones.

<?php
$colores = array("rojo", "verde", "azul");
foreach ($colores as $color) {
    echo "Color: $color <br>";
}
?>

Ejemplo: Lista colores de un array.
