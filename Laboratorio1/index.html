    • Introducción a la Programación Web

    ¿Qué es PHP? (Hypertext Preprocessor) es un lenguaje de programación de código abierto diseñado para el desarrollo web. Es un lenguaje del lado del servidor, lo que significa que el código se ejecuta en el servidor (como Apache en XAMPP) antes de enviar el resultado (HTML, generalmente) al navegador del usuario. Es ideal para principiantes por su sintaxis sencilla y su integración con HTML, bases de datos (como MySQL) y frameworks. PHP es ampliamente utilizado en sitios como WordPress, Joomla y tiendas en línea.

    ¿Por qué usar PHP en desarrollo web?

    * Facilidad de uso: Sintaxis similar a C, fácil de aprender.

    * Integración con HTML: Puedes incrustar PHP dentro de HTML para generar contenido dinámico.

    * Compatibilidad: Funciona con servidores como Apache y bases de datos como MySQL (vía XAMPP, por ejemplo).

    * Comunidad y recursos: Amplia documentación y soporte.

    * Versatilidad: Sirve para crear desde páginas simples hasta aplicaciones complejas.

    Conceptos básicos:

    - PHP se escribe dentro de etiquetas .

    - Los archivos tienen extensión .php.

    En esta página, exploraremos estructuras de control básicas que te ayudarán a controlar el flujo de tu código.

    If

    La estructura if se usa para ejecutar código si una condición es verdadera. Puedes agregar else para alternativas.

    Ejemplo: Verifica si un usuario puede votar basado en su edad.

    Switch

    La estructura switch evalúa una variable contra múltiples casos, ideal para menús o selecciones.

    Ejemplo: Muestra un mensaje según el día de la semana.

    While

    El bucle while repite código mientras una condición sea verdadera.

    Ejemplo: Imprime números del 1 al 5.

    For

    El bucle for es compacto para repeticiones con contador conocido.

    Ejemplo: Cuenta del 0 al 9.

    Foreach

    El bucle foreach itera sobre arrays o colecciones.

    Ejemplo: Lista colores de un array.
